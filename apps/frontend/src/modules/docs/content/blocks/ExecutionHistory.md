# Execution History Block

Every run is recorded in the Execution History, allowing you to track performance over time.

## Key Information

- **Timestamp**: When the test was executed.
- **Model Details**: The specific version of the model used.
- **Result Status**: Success or failure based on your assertions.
- **Latency & Tokens**: Performance metrics for each run.

## Comparison

You can compare current results with historical data to identify regressions in LLM output quality.
